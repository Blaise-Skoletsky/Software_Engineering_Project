Welcome to my game! This game functions similarly to wordle, with the user being able to guess a soccer players nationality, club, and position. Each section is allowed 3 guesses.

Club: If the guess is the same club as the player, the box will be green. If the guess is in the same league as the player, the box will be yellow. If not to either of those, the box will be red.

Country: If the guess is the same country as the player, the box will be green. If the guess is in the same continent, it will be yellow. If the guess is not either, the box will be red.

Position: If the guess is the correct FIFA rating, the box will be green. If the guess is within one rating, it will be yellow. If further than one rating away, it will be red.
